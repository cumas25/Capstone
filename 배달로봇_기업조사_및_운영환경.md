# 배달로봇 기업 조사 및 운영 환경 분석

## 1. 기업 벤치마크: 뉴빌리티 vs 딜리

| 항목 | 뉴빌리티 (뉴비) | 딜리 (배달의민족) |
|---|---|---|
| **모회사** | 독립 스타트업 (2017 창업) | 우아한형제들 (배민) |
| **누적 투자** | 550억원 (2025 시리즈B 251억) | 배민 자체 투자 |
| **로봇 대수** | 305대 운영 중 (2025) | 신형 8월 투입 예정 |
| **운영 거점** | 전국 142개 사이트 | 강남 논현·역삼동 시범 |
| **자율주행 방식** | 카메라 기반 AI + V-SLAM (LiDAR 없음) | 미공개 |
| **핵심 기술** | 맵리스 자율주행, GPS 불필요 | 연석 극복 대형 바퀴, LED 깃대 |
| **적재 용량** | 소형 | 2L 생수 18병 (신형 기준) |
| **속도** | 최대 7.2km/h (보행자 법규 적용) | 미공개 |
| **주행 반경** | 3km 이내 단거리 | 강남 이면도로 중심 |
| **배달 시간** | 20분 내 타겟 | 평균 30분 (강남 기준) |
| **안전 인증** | 국내 최초 실외이동로봇 운행안전인증 | 2025년 6월 인증 획득 |
| **해외 진출** | 일본·사우디·미국 운영 중 | 없음 |
| **재주문율** | 80% (래미안 리더스원 기준) | 만족도 90% ("재이용 의향") |

## 2. 왜 전국 확대가 불가능했나?

### 규제의 벽 (2021년 이전)

- **생활물류법**: "배송은 오토바이·자동차 + 사람만 가능" → 로봇은 배송 주체 자격 없음
- **도로교통법**: 로봇 = '차량'으로 분류 → 보도·횡단보도 통행 불가 → 사실상 도심 운행 전면 차단

### 규제 개선 흐름

| 시점 | 내용 |
|---|---|
| 2021년 이전 | 보도 주행 불가, 규제 샌드박스로만 극히 제한적 실증 |
| 2023년 | 지능형로봇법 개정 → **운행안전인증** 제도 신설 |
| 2024년 | 도로교통법 개정 → 인증 취득 시 **보행자 지위 부여**, 보도 주행 합법화 |
| 2025년 | 실증 범위 **전국 보도로 확대** |

> 핵심: 법 개정 전까지는 인증 제도 자체가 없어서 어느 지역이든 정식 운행 불가 → **지역 문제가 아니라 법 구조 문제**

## 3. 시범운영 지역 선정 기준

공식 발표된 선정 기준은 없으나, 운영 지역의 공통점으로 기준을 역추론:

| 조건 | 판교 | 송도 | 강남 |
|---|---|---|---|
| 계획도시·격자 도로망 | ✓ | ✓ | △ (일부) |
| 평탄 지형 | ✓ | ✓ | △ (이면도로 경사 있음) |
| 고밀도 F&B 수요 | ✓ (테크밸리) | ✓ | ✓ |
| 지자체·정부 협조 | ✓ (자율주행 시범지구 지정) | ✓ | ✓ (로봇 친화도시 선언) |
| 보행자 전용로 확보 | ✓ | ✓ | △ |
| 홍보·레퍼런스 효과 | 중 | 중 | **高** |

> 강남은 지형적으로 불리하지만 **"어려운 환경에서도 된다"는 기술 신뢰성 입증** 목적으로 선택된 측면이 큼

## 요약 인사이트

- **뉴빌리티**: 기술 독립성(LiDAR 없음, 맵리스)과 운영 규모에서 앞서 있으며 해외까지 확장 중
- **딜리**: 배민 플랫폼 연계로 수요 확보가 용이하나, 아직 상용화 초기 단계
- **전국 확대 불가 원인**: 지형이 아닌 **법 구조** 문제였으며, 2024년 도로교통법 개정으로 해소됨
- **시범지 공통 조건**: 계획도시 + 지자체 협조 + 고밀도 수요 → 서울 분석 시 **지자체 협조 여부**도 변수로 추가 필요
